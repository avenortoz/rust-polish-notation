# rust-polish-notation

Simple implementation of inverse polish notation written in rust.

Available operations:
- **-** - substraction
- **+** - addition
- **\*** - multiplication
- **/** - division
- **^** - raising to power

## Build
Copy the repository and run `cargo build`

## Example of usage
```
./rust-polish-notation "(10.7 + 15)^2 - 17"
```
